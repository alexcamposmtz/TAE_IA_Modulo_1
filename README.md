## Descripción del Repositorio

Este repositorio contiene el desarrollo completo de los laboratorios del **Módulo 1: Fundamentos de Programación en Python** del programa de certificación en Inteligencia Artificial 2025. Los laboratorios cubren desde conceptos básicos de programación hasta temas avanzados de Programación Orientada a Objetos, visualización de datos y computación numérica.

---

## Estructura de Laboratorios

### Lab_1: Introducción a Python
**Temas:**
- Instalación de Python y Anaconda
- Configuración del entorno de desarrollo
- Primeros pasos con Python

**Contenido:**
- Guías de instalación
- Configuración de herramientas
- Ejercicios introductorios

---

### Lab_2: Python Básico
**Temas:**
- Variables y tipos de datos
- Operadores
- Entrada y salida de datos
- Conversión de tipos

**Contenido:**
- `ejercicios_basicos.py` - Ejercicios fundamentales de Python
- `deteccion_errores.py` - Práctica de debugging
- Documentación completa

---

### Lab_3: Estructuras de Control
**Temas:**
- Condicionales (if, elif, else)
- Bucles (for, while)
- Control de flujo (break, continue)
- Estructuras anidadas

**Contenido:**
- `ejercicios_control.py` - Ejercicios de estructuras de control
- `deteccion_errores.py` - Errores comunes y correcciones
- Ejemplos prácticos

---

### Lab_4: Modularización
**Temas:**
- Funciones y parámetros
- Módulos y paquetes
- Import y namespace
- Organización de código

**Contenido:**
- `modulo_matematicas.py` - Módulo de operaciones matemáticas
- `modulo_cadenas.py` - Módulo de procesamiento de strings
- `programa_principal.py` - Programa que utiliza los módulos
- `deteccion_errores.py` - Debugging de módulos

---

### Lab_5: Git y GitHub Essentials
**Temas:**
- Instalación y configuración de Git
- Comandos básicos (init, add, commit, push, pull)
- Trabajo con repositorios remotos
- Colaboración en equipo

**Evidencia:**
- Repositorio creado y configurado
- Sistema de control de versiones implementado
- Colaboración entre miembros del equipo

---

### Lab_6: POO I - Clases y Objetos
**Temas:**
- Definición de clases
- Atributos y métodos
- Constructor `__init__`
- Encapsulamiento
- Métodos especiales

**Contenido:**
- `ejercicios_poo.py` - Implementación de clases básicas
- `deteccion_errores.py` - Errores comunes en POO
- Ejemplos de encapsulamiento

---

### Lab_7: POO II - Herencia y Polimorfismo
**Temas:**
- Herencia simple y múltiple
- Método `super()`
- Sobrescritura de métodos
- Polimorfismo
- Clases abstractas

**Contenido:**
- `ejercicios_herencia.py` - Jerarquías de clases
- `deteccion_errores.py` - Debugging de herencia
- Ejemplos de polimorfismo

---

### Lab_8: POO III - Composición y Agregación
**Temas:**
- Composición vs Herencia
- Agregación
- Relaciones entre objetos
- Diseño orientado a objetos

**Contenido:**
- `ejercicios_composicion.py` - Implementación de relaciones
- `deteccion_errores.py` - Errores en diseño OOP
- Patrones de diseño básicos

---

### Lab_9: POO IV - Temas Avanzados
**Temas:**
- Métodos estáticos y de clase
- Properties y decoradores
- Context managers
- Generadores e iteradores
- Métodos mágicos avanzados

**Contenido:**
- `ejercicios_avanzados.py` - Técnicas avanzadas de POO
- `deteccion_errores.py` - Debugging avanzado
- Ejemplos de decoradores y generadores

## Objetivos de Aprendizaje

Al completar estos laboratorios, se adquierieron las siguientes competencias:

1. Fundamentos sólidos de programación en Python
2. Dominio de estructuras de control y modularización
3. Comprensión profunda de Programación Orientada a Objetos
4. Capacidad para diseñar jerarquías de clases eficientes
5. Habilidades de debugging y detección de errores
6. Visualización efectiva de datos con Matplotlib
7. Análisis numérico con NumPy
8. Uso profesional de Git y GitHub
9. Mejores prácticas de documentación de código
